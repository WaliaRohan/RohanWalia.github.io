Academic Projects

A highlight of some of the projects I have worked on over the course of my Masters and PhD so far.

AI for Robotics

Neural Network Control Allocation for Ship Thrusters

Aug 2024 – Dec 2024
Control Allocation separates high-level control commands from low-level control system architecture. In this project, implemented a multi-layer perceptron encoder–decoder network that maps force and moment requests to thruster commands for dynamic positioning of autonomous ships. Designed a composite loss enforcing command fidelity, thruster saturation limits, and power minimization.

LSTM-Based Trajectory Prediction for 3D Dubins Paths

Aug 2024 – Dec 2024
Designed an LSTM-based network for generating 3D Dubins-compliant trajectories using only the initial pose, goal position, and climb angle. Trained with MSE loss on a custom synthetic dataset of 432,000 variable-length trajectories.

GitHub

Path planning and Navigation

Adaptive Drone Navigation

March 2024
Developed a C++ ROS 2 package for PX4-based aerial vehicles enabling autonomous and sequential traversal of trajectory waypoints. Implemented dynamic obstacle avoidance with shortest return to the designated route and engineered a finite-state machine for navigation, avoidance, and expedited path re-entry. Designed for PX4 SITL and Gazebo, with extensibility to real hardware.

GitHub

Path Planning Algorithms

Jan 2021 – May 2021
Implemented classical graph- and sampling-based path planners including Dijkstra, A*, ANA*, RRT, RRT*, and Bi-directional RRT*. Image-based planners take a map image, start location, and goal location, and compute collision-free paths using binary occupancy grids derived via OpenCV. Planner performance was evaluated under increasing map sizes and obstacle densities.

GitHub

Controls and Dynamics

LQR Control of a Quadcopter

Aug 2020 – Dec 2020
Implemented a Linear Quadratic Regulator (LQR) for a 6-DOF quadcopter in MATLAB to stabilize hover and track trajectories. The controller was designed using a linearized quadrotor dynamics model to optimize the trade-off between thrust performance and battery consumption.

GitHub

Custom Differential Drive Robot

Aug 2020 – Dec 2020
Designed and constructed a custom differential-drive mobile robot using a Raspberry Pi, four DC motors, and an L293 motor driver. Established SSH-based remote access for control, monitoring, and debugging, and implemented Python scripts to translate velocity trajectory setpoints into PWM duty-cycle commands for motor actuation.
